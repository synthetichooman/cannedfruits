# Changelog

## v0.2n

Initial native macOS release candidate.

### Added

- SwiftUI native macOS app.
- CannedFruits icon in the app sidebar.
- Single-seller FruitsFamily storefront project generator.
- Seller URL setup and seller resolution.
- Design memo, reference URL, accent/background/text color settings.
- Logo import with generated web logo and favicon assets.
- Browser-based local preview.
- Project folder export for GitHub and Cloudflare Pages.
- LLM connection guide and copyable LLM harness.
- Deployment guide with LLM prompt sample.
- Post-deployment recommendations.
- Manual deployment guide for users who do not use an LLM.

### Web Template

- Dynamic Cloudflare Pages Functions architecture.
- Product list and product detail pages.
- FruitsFamily CTA links.
- Sold status display.
- Required unofficial-tool disclaimer.
- Cleaner product detail layout.

### Distribution Note

This build is not signed with an Apple Developer ID and is not notarized. Users may need to right-click the app and choose Open on first launch.
