# 설치 안내

## 1. 다운로드

GitHub Releases에서 최신 버전의 `CannedFruits-v0.2n.dmg`를 다운로드합니다.

DMG가 열리지 않으면 `CannedFruits-v0.2n-macOS.zip`을 다운로드해 압축을 풉니다.

## 2. 앱 옮기기

DMG를 열고 `CannedFruits.app`을 `Applications` 폴더로 옮깁니다.

ZIP을 받은 경우 압축을 푼 뒤 `CannedFruits.app`을 `Applications` 폴더로 옮깁니다.

## 3. 처음 실행

이 앱은 아직 Apple Developer ID로 서명하거나 notarization 처리하지 않았습니다.

macOS가 경고를 띄우면 아래 방식으로 여세요.

1. `Applications` 폴더에서 `CannedFruits.app`을 찾습니다.
2. 앱을 우클릭합니다.
3. `열기`를 선택합니다.
4. 다시 한 번 `열기`를 선택합니다.

처음 한 번만 이렇게 열면 이후에는 보통 더블클릭으로 열립니다.

## 4. 사용 흐름

1. 새 프로젝트를 만듭니다.
2. FruitsFamily 셀러 링크를 입력합니다.
3. 샵 이름, 색상, 로고, 디자인 메모를 저장합니다.
4. 브라우저에서 미리보기를 확인합니다.
5. 완성된 프로젝트 폴더를 내보냅니다.
6. Codex나 Claude에 폴더를 연결하거나, 앱의 수동 배포 안내를 따라 GitHub와 Cloudflare Pages에 연결합니다.

## 5. 주의

CannedFruits는 FruitsFamily와 관련이 없는 비공식 독립 도구입니다. 사용 과정에서 발생하는 계정, 운영, 정책상 불이익은 사용자에게 귀속됩니다.
