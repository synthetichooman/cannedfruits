CannedFruits

FruitsFamily 셀러를 위한 독립 웹샵 제작 앱입니다. 셀러 링크와 디자인 정보를 입력하면 GitHub와 Cloudflare Pages에 배포 가능한 웹사이트 프로젝트 폴더를 만듭니다.

v0.2n 다운로드 GitHub 저장소

설치 전 확인

  • macOS 14 이상에서 사용합니다.
  • 현재 빌드는 Apple Developer ID로 서명되거나 notarization 처리되지 않았습니다.
  • 처음 실행할 때 경고가 나오면 앱을 우클릭하고 열기를 선택하세요.

기본 흐름

  • 앱에서 새 프로젝트를 만듭니다.
  • FruitsFamily 셀러 링크, 샵 이름, 색상, 로고를 입력합니다.
  • 브라우저에서 미리보기를 확인합니다.
  • 완성된 프로젝트 폴더를 내보냅니다.
  • 내보낸 폴더를 GitHub와 Cloudflare Pages에 연결합니다.

고지

CannedFruits는 FruitsFamily와 관련이 없는 비공식 독립 도구입니다. 사용 과정에서 발생하는 계정, 운영, 정책상 불이익은 사용자에게 귀속됩니다.
